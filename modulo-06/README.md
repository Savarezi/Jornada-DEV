## 🎯 Objetivos de aprendizagem

- Criar e utilizar funções.
- Dividir programas em partes menores e reutilizáveis.
- Diferenciar funções de procedimentos.
- Trabalhar com arrays e arrays multidimensionais.
- Utilizar arrays como estruturas de dados.
- Criar funções para manipular arrays.
- Utilizar loops em conjunto com funções e arrays.
- Organizar código em bibliotecas separadas.
- Desenvolver um sistema completo em memória.
