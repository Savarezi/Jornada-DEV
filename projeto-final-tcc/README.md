# 🏗️ Projeto Final (TCC) — Controle de Não Conformidades no Protheus

## 📌 Sobre o projeto

Este projeto será desenvolvido como trabalho final da jornada **Start+ TOTVS Paulista**.

O objetivo é construir uma funcionalidade real dentro do Protheus para controle de **não conformidades na entrada de materiais**, simulando um cenário corporativo baseado em processos de qualidade industrial.

---

## 🎯 Cenário

A **Indústria XYZ** precisa registrar problemas encontrados no recebimento de materiais para manter conformidade com a certificação **ISO 9001**.

Exemplos de não conformidades:

- Material fora da especificação
- Falta de certificado de qualidade
- Produto com defeito
- Divergência de quantidade
- Problemas de embalagem

---

## ⚙️ O que será desenvolvido

Este projeto terá:

- Criação de tabelas customizadas via **SX2** e **SX3**
- Cadastro de fornecimentos
- Cadastro de ocorrências de não conformidade
- CRUD completo (inclusão, consulta, alteração e exclusão)
- Integração com menus no módulo **SIGACOM**
- Regras de negócio com validações e gatilhos
- Relatório gerencial de acompanhamento
